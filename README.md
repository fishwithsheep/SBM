# A structure-preserving collisional particle method for the Landau kinetic equation

SBM is a structure-preseving perticle method for Landau kinetic equation, whose computation cost is just O(N).

Here we offer the code of four numerical experiments in our article, including EM method and SBM method. Also we offer the npy file of the initial sampling data and sampling program. For language we choose Python. 

Authors of this article is Kai Dua from Fudan University, Lei Li from Shanghai Jiao Tong University, Yongle Xie from Fudan University, Yang Yu from Shanghai Jiao Tong University.

## Features

- Structure-conserving
- Computation cost is O(N)
